Ann Arbor DDA has a new web site, and real time parking information is available again

The Ann Arbor DDA has launched a new web site, and one of the features of it is a map with information available in real time about parking availability at various structures. The structures are in a shade of pale blue, and the lots are in pale green; my current location is marked with a bubble. If you're looking for just space available information, there's a JSON file with that data; you'll have to unpack it to map structures to their ID numbers. »
