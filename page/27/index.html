Joyent Manta: map/reduce for Unix natives

Joyent has released Manta, a computing environment that includes a powerful map/reduce system which leverages native Unix commands. Unlike a Hadoop world where you have to spin up a Java environment to get anything done, Manta makes map/reduce as simple as possible for those of us who know the Unix shell pretty well. Some examples may be useful. for n in $HOME/doc/books/* do wc -l $n >> $TMP/wc-tmp done awk '{ t += $1 } END { print t }' < $TMP/wc-tmp has an equivalent in Manta as something like mfind -t o /$MANTA_HOME/stor/books | \ mjob create -o -m "wc -l" -r "awk '{ t += \$1 } END { print t }' " where the word count command "wc" is the "map" step, and the awk summation is the "reduce" step. »
