Programming your new radio with CHIRP

From the CHIRP web site - CHIRP is a free, open-source tool for programming your amateur radio. It supports a large number of manufacturers and models, as well as provides a way to interface with multiple data sources and formats. The basic idea is that rather than learning the peculiarities of your radio's programming interface, you use CHIRP which has access to databases like Radio Reference and RepeaterBook to find the channels you want to have easy access to. »
