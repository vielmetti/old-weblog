Learning Morse Code

I’m trying to learn Morse Code, at least to be able to copy characters at a a fast enough pace to understand what the IDs on the local 2 meter amateur repeaters are saying. Over a previous holiday break I had sorted out at least a little bit of sending Morse, so some letters are already familiar (CQ, K, E, SOS). I don’t yet know how to either send or recognize my call sign. There’s a ways to go.

Ward Cunningham K9OX has a page full of ham radio software, including a Morse Code Teaching Machine that does a nifty adaptive learning system to feed you the letters you need work on. It dates from the article A Fully Automatic Morse Code Teaching Machine, QST (May 1977). Ward wrote it to fit on a very tiny machine, and it’s been subsequently ported and improved to run on bigger machines, but still at its heart it’s old and tiny (and in some essential way very wonderful).

I’m using the Macintosh version; there’s an interesting version for Ward’s federated wiki as well, which would be worth trying.

73 de Ed KD8OQG
