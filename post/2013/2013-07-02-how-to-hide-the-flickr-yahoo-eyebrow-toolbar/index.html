How to hide the Flickr Yahoo! "eyebrow" toolbar

How to hide the Flickr Yahoo! "eyebrow" toolbar

Flickr users are not happy.

A Greasemonkey script to remove the Yahoo eyebrow uses Javascript. A key element reads

        document.getElementById('eyebrow').style.display = 'none';

A second approach uses Adblock Plus to wipe out the ads for Yahoo services; note, you might have to buy into a lot more than just blocking this ad if you go that route.

A third route uses CSS to hide the offending elements; I saw it fly past this a.m. but can't find it for the life of me. Anyone? UPDATE: Thanks @davidblivens and @stephaniefysh, I found a Stylish script at userstyles.org, with the key element being

div#eyebrow {display: none;}

Stylish is downloadable for Chrome and Firefox at userstyles.org, if you need it.
