some details of setting up Movable Type: day 13

some details of setting up Movable Type: day 13

public notes from day 13 at AnnArbor.com.

I'm noting some MT setup details here, since I need a spot for it, and since I've been relatively spoiled by having the Typepad crew figure out how to make things just work.


1.  "Replace Word Characters".  A default setting when you paste in Word documents is to have them paste in with entity characters; this setting converts those to ASCII.  (Word, bleah, but it's the norm.)

2.  EntryIfExtended.  A template tag to use to conditionally include a "more..." button at the bottom of a page or entry, to link to the whole thing.  (Typepad doesn't do extended entries.)

3.  Creating comment feeds.  A forum post showing a couple of wrong ways and a right way to do it.

more to come, but that's tonight; spelunking through the docs to see what's possible.
