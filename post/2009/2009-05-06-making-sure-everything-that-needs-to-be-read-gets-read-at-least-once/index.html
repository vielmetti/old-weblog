making sure everything that needs to be read gets read at least once

I'm always behind on email, but this time it's worse than normal. 

Here's a catching up strategy.  To do this you need to know that anything I think might need a response gets a star in Gmail, but sometimes also gets archived for later.

1.  Move all starred items to the inbox
2.  Mark all inbox items as new
3.  Marvel that there are 238 things to read that haven't been touched
4.  Lean on the j key to zoom through them quickly
5.  Count only the number that hasn't been looked at at least once

I'm also using tags in Gmail, so an additional strategy is to select all the messages, look in the labels tab, see which labels look weird, unselect everything, and then search just for the label you found (to deal with it or delete it).

This isn't so much a productivity hack as it is a despiration hack to make sure that of the things that have been overtaken by events, that they at least don't consume mind-space making me think that I need something more to do.
