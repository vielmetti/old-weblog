28 June 2015

Running DD-WRT as a switch on a Linksys WRT54GL

I don't really need more wifi access points in the house, but I do need more wired ethernet ports. Thanks to ellemef's box o'goodies I have a WRT54GL to play with, and I put DD-WRT on it set up in a configuration to run as a switch plugged into my existing ATT UVerse router.

I got the DD-WRT firmware loaded by following the provided directions. Then I followed the instructions at You are here: DD-WRT wiki mainpage / Linking Routers / Switch for configuration. A quick reboot and some recabling and a couple of Raspberry Pis now have their own ethernet ports.

The WRT54GL has a tiny amount of memory by modern standards - 4 megabytes of flash, and only 16 megabytes of system memory - but that's fine for the software of its era.
