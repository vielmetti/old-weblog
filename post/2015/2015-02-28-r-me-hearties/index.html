R, me hearties

I've been fond of maps and graphs for a long time, but haven't had a good set of tools to play with that easily generate decent versions of same from various data sets I've found or accumulated.

The small hope is that a little bit of hacking on the right tools will fix this.

One of the tools of choice is R, a language for statistics and data analysis. It's tremendously powerful and a bit complicated, but I'm not trying to learn the whole thing; rather, I'm after just the barest minimum so that I can competently generate a scatterplot of data and place it onto a map.

I'm working through the examples on Zev Ross's Mapping in R using the ggplot2 package, which looks like about the right level of detail to start from. The other useful and helpful information that I'm bootstrapping from is Jeroen Janssens Data Science at the Command Line

The first production of this is just a scatterplot; it's based on Ann Arbor Downtown Development Authority parking data.
