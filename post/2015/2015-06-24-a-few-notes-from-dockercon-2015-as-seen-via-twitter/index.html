a few notes from Dockercon 2015, as seen via Twitter

The annual conference for Docker, Dockercon, has been going on. A few highlights from the event as seen through Twitter - I was not there to attend it.

Docker is real on the Raspberry Pi through support for the ARM processor.

Awesome live demo of Docker on a Raspberry Pi by @Quintus23M in the #dockercon closing keynote. pic.twitter.com/DJfBtpcYws

— Bridget Kromhout (@bridgetkromhout) June 23, 2015

More details from the presenter, Hypriot-Demo and challenge at DockerCon 2015.

Docker's plumbing is being spun out as runC. The new infrastructure supports migration of containers from one server to another, illustrated through a Quake III demo.

QuakeIII in a container, migrated seamlessly from Singapore to Amsterdam to London using #runc. #dockercon pic.twitter.com/no3c0G1MDY

— Paul V. Novarese (@pvnovarese) June 24, 2015

For more highlights, see the Venturebeat roundup of Dockercon 2015.
