restaurant inspections

Every month the Washtenaw County Health Department does a round of restaurant inspections. Wayne Eaker built an importer that pulls in data from the site that hosts the official county inspections, and is storing the inspection data on Github to make it easier for the next person who wants to build a nice interface to all the data.

Wayne was good enough to find a spot to host an interface to this data that lets you browse through it; you can find it at

http://food-inspections.annarbortelegraph.com

There are permalinks to each inspection so that you can link to them. See e.g. recent inspections for

  • Slurping Turtle, June 17, 2015
  • China Chef, Ypsilanti, April 27, 2015
  • Nu Wave Fish and Chicken, June 22, 2015

for three examples of places where I want to see a clean inspection on a followup visit before I go or go back to there.
