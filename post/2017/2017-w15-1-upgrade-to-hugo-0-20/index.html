Upgrade to Hugo 0.20

Apr 10, 2017

I’ve upgraded this weblog to Hugo 0.20, which is available by Homebrew on the Mac and with a series of prebuilt binaries for many systems.

Hugo in this 0.20 new release promises a lot of new features, including but not limited to support for rendering calendars. I’ll give it a try.

One of my goals of Hugo is getting it to run on linux/arm64. It should be easy, since it’s written in Go, and golang ports are absurdly easy. I didn’t see how they’re doing builds immediately when I looked at the repo so I opened an issue.

Hugo fixed one bug, and the bug fix caught me. Make sure that you add a trailing / to your URL if you need a trailing slash in your names, because that is no longer optional in BaseURL.
