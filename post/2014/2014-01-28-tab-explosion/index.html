Tab explosion!


I'm (still) using OneTab to keep control of my tabs in Chrome. I wrote about it previously; it's an extension that suspends the processes of tabs you mark as inactive, so that they can be restored quickly. As a bonus, it's good at keeping things out of working memory, so you can have a system with "only" 2 gigabytes and still run Chrome without endless swapping.

The Internet, in its infinite wisdom, has a lot of things competing for our attention. The only problem is that it's easy to save a tab group but as hard as ever to really mark a tab as closed. I'm sure there's some stable working set that I should keep going, but it's also satisfying to decrease the n by closing tabs.
