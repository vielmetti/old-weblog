6 November 2014

FOIA: A2 Fix It request, Issue ID: 1275000, Service Request ID: 272504 regarding traffic light at Pauline and Maple

To: Ann Arbor City Clerk Date: November 5, 2014

FOIA

Please provide the following records:

  1. Records within the A2 Fix It / SeeClickFix system regarding Issue ID 127500, Service Request ID 272504, reported on September 8, 2014.

  2. Records within the city's Cityworks system related to this same service request.

  3. Any electronic mail communications associated with this service request not captured within A2 Fix It or Cityworks.

  4. Maintenance and service records for the traffic signal and associated systems at the corner of Pauline and Maple, specifically those related to the maintenance of, placement of, and plans for repair of the vehicle sensors at this location, believed to be records held by the Signs and Signals unit and associated with this issue.

Thank you for your prompt attention to this request. As these records should be readily available, please inform me prior to fulfilling the request to notify me and get my approval if there are to be any costs related to this request. Please provide records in electronic format to the extent that this is possible.

Thanks again

Ed
