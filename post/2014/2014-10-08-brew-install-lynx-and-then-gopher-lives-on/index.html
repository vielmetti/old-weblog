"brew install lynx", and then Gopher lives on

Since we're reinventing the Olde Web with systems like tilde.club, here's a screen shot of the current state of Gopher.

"brew install lynx" gets a Lynx client on my Mac, and then you point lynx at gopher://gopher.floodgap.com and you're good to go.

"ssh new@sdf.org" and you can set yourself up a shell account with a gopher client installed and gopher server space available.
