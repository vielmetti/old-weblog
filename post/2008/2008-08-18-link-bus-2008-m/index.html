Link Bus 2008 map (plus a sticker project)

18 August 2008

Link Bus 2008 map (plus a sticker project)

The Ann Arbor Link Bus starts service again next week.  Every year I get tired of not being able to figure out when the bus is going to show up, and this year I tried to build a new map to help out.  This is a work in progress.

The key idea here (so far poorly executed, but I'll get it right) is to replace the stop letters with stop numbers, and to number the stops by the number of minutes after the quarter hour that the bus is expected.  So the Y stop becomes an 11 stop, since the bus is supposed to get there at 11 26 41 56.

Google Maps now has a way to do custom icons, and I'm using letters from a Unicode alphabet to get the circled numbers.  A few glitches still until I host those icons locally.  It's hard to get the icons located precisely right at all scales and resolutions, don't know how to fix that (short of actually doing paste up on a paper map).

UPDATE: Dave Askins is contributing a set of icons for the stop markers - the “9” stop there is the new and improved, which allows for better stop locating.
