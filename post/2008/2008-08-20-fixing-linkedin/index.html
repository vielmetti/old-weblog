Fixing LinkedIn for Groups with Greasemonkey; step one, the naming of the parts

20 August 2008

Fixing LinkedIn for Groups with Greasemonkey; step one, the naming of the parts

I use LinkedIn for Groups to manage the Cisco Alumni Association group.  One of the qualification steps is that you have to have worked for Cisco in the past (and you'd be surprised how many people apply who haven't any connection at all).

The admin UI for that system is clumsy if you have to approve and delete a lot of people - basically there's no built-in support for checking someone out short of opening up a whole new window with their profile.  That makes the workflow very clumsy and slow.

The fix-it-yourself approach to this is Greasemonkey, splicing in the existing code that does this feature on other parts of the site to the Admin UI page.  for your (really, for my) reading enjoyment, here's that code.

Part two is causing this code to be called on another page other than the one it currently is being called!

 

* js/miniprofile_popup.js */

var miniProfile=function()
{var profileLinks,popContainer;
var responseSuccess=function(o){popContainer.getElementsByTagName('div')[0].innerHTML=o.responseText;
popContainer.setAttribute('hasMiniPanel','true');
lui.miniPanel.setBody(o.responseText);
addPanelEvents();
lui.miniPanel.move(popContainer);
lui.miniPanel.show();}

var callBack={success:responseSuccess,argument:[popContainer],timeout:10000}

var addPanelEvents=function()
{lui.miniPanel.addCloseEvent();
var actionForms=YAHOO.util.Dom.getElementsByClassName('action-form','div',profilePanelBody);
if(actionForms=="")return;
for(var i=0;actionForms.length>i;i++){actionForms[i].style.display="none";}
var flagLink=YAHOO.util.Dom.getElementsByClassName('action-flag','a',profilePanelBody);
YAHOO.util.Event.addListener(flagLink,'click',function(e){YAHOO.util.Event.preventDefault(e);
toggleFlagForm(actionForms);},profilePanelBody);}

var toggleFlagForm=function(el)
{for(var i=0;el.length>i;i++)
{if(el[i].style.display=="block")
{lui.hide(el[i]);}else{lui.show(el[i]);}}}
return{init:function(){profileLinks=YAHOO.util.Dom.getElementsByClassName('miniprofile-container','span','body');
lui.miniPanel.init(profileLinks,miniProfile.showProfile,miniProfile.fetchProfile);
profilePanelBody=lui.miniPanel.get();},fetchProfile:function(el){popContainer=el;
var url=el.className.split(' ')[1];
YAHOO.util.Connect.asyncRequest('GET',url,callBack);},
showProfile:function(el){popContainer=el;lui.miniPanel.setBody(el.getElementsByTagName('div')[0].innerHTML);
addPanelEvents();
lui.miniPanel.move(el);
lui.miniPanel.show();}}}();

miniProfile.init();
