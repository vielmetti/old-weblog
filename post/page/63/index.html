IPv6 over social network; a Facebook application for routing packets over friend networks

noted on Geert's blog IP over Social Network is a Facebook application which allows any Facebook member to create its own IPv6 virtual router for free and build an IPv6 network over the Facebook Social Network! Once you have created your virtual router by installing this free application, you can then add up to 12 interfaces (i.e. links) connected to your other friends’ routers. IPv6 packets can then be exchanged over those links and of course, like in all IP networks, can be routed to any destination over the network (including a special gateway to the IPv6 Internet of course)! »
