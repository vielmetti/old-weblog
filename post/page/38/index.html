ArborWiki is moving to a Localwiki platform; what to expect

Arborwiki is a local encyclopedia about the Ann Arbor area that anyone can edit. Up til now, it's been running on the same Mediawiki software that powers Wikipedia, but we're in the process of moving it to a new Localwiki instance. Localwiki is wiki software designed especially for using in city wikis. It's based off of ideas learned from DavisWiki, a very successful wiki based in Davis, California. The Localwiki software is especially good at handling maps, and so it should be that much easier to map out the streets and roads of Ann Arbor and surrounding Washtenaw County. »
