Olivetti Programma 101

The first computer that I ever had all to myself was the Olivetti Programma 101. This was a 1965-era Italian machine that was just a little more than a programmable calculator. It used flexible magnetic cards for program storage, discrete logic for all of its computing functions, delay line memory (240 bytes), and it all sat in a 65 pound package. Output was on a humming little printer built into the machine. »
