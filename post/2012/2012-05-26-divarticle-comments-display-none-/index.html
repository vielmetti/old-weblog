div.article-comments { display: none; }

May 26, 2012

A little bit of CSS makes reading certain newspaper web sites a lot easier.

div.article-comments {
   display: none;
}

Insert this into your favorite site using a tool like Stylist for Chrome, or Stylish for Firefox.
