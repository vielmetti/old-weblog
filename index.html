Installing Octopress on MacOS X 10.6.8

Aug 11th, 2013

Started to install Octopress on my Mac, but got into a problem where one of the dependencies wanted a later version of ruby (or at least a later patchlevel) than the one I had just built. That’s enough to stop me for now.

But not for very long!

This Stack Overflow post gives the syntax for installing a particular patchlevel of ruby, and so I did it.

Then it came to creating a repo on github, and I got confused; should the URL be vielmetti.github.com or vielmetti.github.io ? Because one part of the doc used one name and the in-line code example used another. I had a skew between the two, and now it looks like things are hopelessly confused (or at least confused enough that I should delete a repo and start over).
