Command Line JSON Tools

Aug 13th, 2013

I’m on the lookout for tools that I can invoke from the command line that are wise to the JSON file format.

So many Unix tools assume that there’s one record per line, and are less than useful when dealing with a blob of JSON. The objective here is to be able to do non-trivial JSON transformations without dropping into a programming language or an editor to do the task.

What I’ve found and installed so far:

  • Antonin Hildebrand’s csv2json gem, written in Ruby; it converts CSV files to JSON, or JSON files to CSV.
  • Marc Lehmann’s JSON-XS command line utility, which pretty-prints JSON and converts to and from several file formats; in Perl, with an interface that lets you apply a perl command to each JSON object
  • Kyle Keen’s jshon, an ambitious parser and processor for JSON files that is designed for command line use; built on top of Jansson.
  • Micha Niskin’s jsawk, “like awk, but for JSON”; lets you apply a Javascript transform to each JSON object as it streams by
  • Dave Dopson’s underscore-cli, a “command-line utility belt for hacking JSON”, with functions in Javascript using the underscore.js JSON library

Chances are good that one or the other of these packages will get you one step further than you are if you had been trying to manipulate JSON with sed and awk!
